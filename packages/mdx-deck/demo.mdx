## Embed

You can Embed an MDX Deck presentation anywhere in React

---

```jsx
import React from 'react'
import { Embed } from 'mdx-deck'
import MyDeck from './my-deck.mdx'

export default props =>
  <Embed
    src={MyDeck}
    slide={4}
  />
```
